# Quizer 

Yet another 'Quiz Management System' Application bulid just for college sake.

## Tech Stack
- Java Spring Boot
- JDBC
- Next.js 
- PostgreSQL
- JDBC

## Pre-requisites
- Node.js 
- Java 11
- Maven

## Installation
```bash
git clone https://github.com/kmr-ankitt/quizer.git
cd quizer
```

## Running the Application

### Frontend
```bash
cd client
pnpm install
pnpm dev
```
### Backend
```bash
cd server
mvn spring-boot:run
```
